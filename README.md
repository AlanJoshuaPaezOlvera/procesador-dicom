# Procesador DICOM

Software académico desarrollado en Python para cargar, visualizar y procesar imágenes médicas en formato DICOM.

## Funciones
- Carga de archivo DICOM por explorador
- Carga por ruta manual
- Visualización de imagen original y procesada
- Operaciones aritméticas
- Operaciones lógicas
- Filtrado espacial
- Procesamiento en frecuencia (FFT/IFFT)

## Requisitos
Instalar dependencias con:

pip install -r requirements.txt

## Ejecución
python dicom_processor_app.py
